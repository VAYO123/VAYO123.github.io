CEIT Network
ສ້າງຕັ້ງຂື້ນໃນປີ 2023 ເຊີ່ງຫາກາສ້າງຕັ້ງຂື້ນມາບໍ່ດົນນີ້ເອງ ມີໜ້າທີ່ເຮັດວຽກກ່ຽວກັບການເຊື່ອມຕໍ່ ແລະ ສົ່ງຂໍ້ມູນລະຫວ່າງອຸປະກອນຫຼາຍໆໃນເຄືອຂ່າຍ ເຊັ່ນ ຄອມພິວເຕີ, ໂທລະສັບ, server, printer ແລະອຸປະກອນອື່ນໆ ໂດຍມີຫຼາຍໜ້າທີ່ດັ່ງນີ້:
ໜ້າທີ່ຫຼັກໆຂອງ Network
1. ສົ່ງແລະຮັບຂໍ້ມູນ (Data Communication)
• ຊ່ວຍໃຫ້ອຸປະກອນສອງຫຼືຫຼາຍໆ ແລກປ່ຽນຂໍ້ມູນກັນໄດ້ (ເຊັ່ນ ສົ່ງອີເມວ, ໂອນໄຟລ໌)
2. ແບ່ງປັນຊັບພະຍາກອນ (Resource Sharing)
• ໃຊ້ printer ຫນຶ່ງເຄື່ອງຮ່ວມກັນຫຼາຍເຄື່ອງ, ແບ່ງໄຟລ໌ກັນໃຊ້
3. ເຂົ້າເຖິງຂໍ້ມູນລວມ (Centralized Data Access)
• ຂໍ້ມູນທັງໝົດຖືກເກັບໄວ້ທີ່ server ການຈັດການຈຶ່ງງ່າຍຂື້ນ
4. ຄວາມປອດໄພໃນຂໍ້ມູນ (Security Control)
• ຈັດການຜູ້ໃຊ້, ສິດໃນການເຂົ້າເຖິງ, ການເຂົ້າລະຫັດຂໍ້ມູນ
5. ການຄວບຄຸມ ແລະ ການຈັດການ (Monitoring & Management)
• ການຕິດຕາມສະພາບຂອງເຄືອຂ່າຍ ແລະ ການແກ້ໄຂບັນຫາ
ໜ້າທີ່ເຮັດວຽກຂອງ CEIT Network
1. ເຊື່ອມຕໍ່ຄອມພິວເຕີ ແລະ ອຸປະກອນຕ່າງໆ ເຊັ່ນ ໂທລະສັບ, printer, server ໃຫ້ສາມາດສື່ສານແລະແບ່ງປັນຂໍ້ມູນກັນໄດ້
2. ສົ່ງ ແລະ ຮັບຂໍ້ມູນ ເຊັ່ນ ການສົ່ງອີເມວ, ໂອນໄຟລ໌, ແຊັດ, ເຊື່ອມຕໍ່ເວັບໄຊ
3. ໃຊ້ຊັບພະຍາກອນຮ່ວມກັນ ເຊັ່ນ printer ຫນຶ່ງເຄື່ອງ ໃຫ້ຄອມພິວເຕີຫຼາຍເຄື່ອງໃຊ້ກ່ຽວກັນໄດ້
4. ການຄວບຄຸມ ແລະ ຄວາມປອດໄພ ປ້ອງກັນການເຂົ້າເຖິງຂໍ້ມູນທີ່ບໍ່ອະນຸຍາດ, ຕິດຕັ້ງ firewall ແລະ system monitoring
5. ສື່ສານແບບທັນເວລາ (Real-time) ເຊັ່ນ ການໂທຜ່ານວີດີໂອ, ການ Live stream, ແຊັດອອນໄລນ໌
CEIT Network ຄວາມຮູ້ກ່ຽວກັບ Network ຄື
1. ຄວາມເຂົ້າໃຈກ່ຽວກັບການສື່ສານຂໍ້ມູນ: ,ການສົ່ງ/ຮັບຂໍ້ມູນເກີດຂຶ້ນຢ່າງໃດ ,Protocol ແຕ່ລະຢ່າງເຊັ່ນ TCP/IP, HTTP, FTP
2. ພື້ນຖານການອອກແບບ ແລະ ການຈັດຕັ້ງເຄືອຂ່າຍ: ,ການກຳນົດ IP address ,Subnetting ,ການຕິດຕັ້ງ Switch, Router, Hub
3. ດ້ານຄວາມປອດໄພຂອງຂໍ້ມູນ: ຮູ້ຈັກ Firewall, VPN, Antivirus, ການເຂົ້າລະຫັດ (Encryption)
4. ການແກ້ໄຂປັນຫາເຄືອຂ່າຍ (Troubleshooting): ,Ping, Traceroute, Wireshark ,ການວິເຄາະບັນຫາການເຊື່ອມຕໍ່
5. ປະເພດຂອງເຄືອຂ່າຍ: .LAN, WAN, WLAN, MAN .Client-server, Peer-to-peer
