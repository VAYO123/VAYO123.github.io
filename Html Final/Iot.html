CEIT IoT Lab
ສ້າງຕັ້ງຂື້ນໃນປີ 2019 ແມ່ນກຸ່ມກິດຈະກໍາ ທີ່ຕັ້ງຢູ່ໃນພາກວິຊາວິສະວະກໍາຄອມພິວເຕີ ແລະ ເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານ ເຊິ່ງສຶກສາຄົ້ນຄວ້າກ່ຽວກັບ IoT (Internet of things) ຫຼື ກໍຄືໃຊ້ອຸປະກອນ ແລະ ສິ່ງຕ່າງໆເຊື່ອມຕໍ່ກັບອິນເຕີເນັດ. ເຊິ່ງສາມາດຄວບຄຸມການນໍາ ໃຊ້ອຸປະກອນຜ່ານເຄືອຂ່າຍອິນເຕີເນັດ ເຊັ່ນ: ການເປີດ - ປິດ ຄວບຄຸມເຄື່ອງໃຊ້ໄຟຟ້າຢູ່ເຮືອນໂດຍເຊື່ອມຕໍ່ອຸປະກອນຄວບຄຸມ.
ຫນ້າທີ່ວຽກຂອງ CEIT IoT Lab
• ການພັດທະນາ Hardware
• ການພັດທະນາ Software
• ການດູແລລະບົບ ແລະ ບໍາລຸງຮັກສາ
• ການຈັດການເອກະສານ ແລະ ປະສານງານ
• ການພັດທະນາບຸກຄະລາກອນ
CEIT IoT Lab ໃຫ້ຄວາມຮູ້ຫຼັກໆແມ່ນກ່ຽວກັບ IoT (Internet of Things) ທີ່ແມ່ນອຸປະກອນ ແລະ ສິ່ງຕ່າງໆເຊື່ອມຕໍ່ກັບອິນເຕີເນັດ ແລະ ຄວມຄຸມໄດ້ໂດຍມີອົງປະກອບຄວາມຮູ້ຍ່ອຍ:
• ຄວາມຮູ້ທາງດ້ານ Hardware (ການຕໍ່ວົງຈອນ, ການນຳໃຊ້ sensor ຕ່າງໆໃຫ້ຖືກຕ້ອງ, ການອອກແບບວົງຈອນໄຟຟ້າຕ່າງໆ...)
• ຄວາມຮູ້ທາງດ້ານ Software (ການຂຽນໂຄ໊ດແລະ ກຳນົດການເຮັດຕ່າງໆໃຫ້ກັບບ໋ອດ microcontroller, ການອອກແບບໂລຈິກຂອງໂຄ໊ດຕ່າງໆເພື່ອ runtime ມີປະສິດທິພາບ, ການຂຽນເວັບ ຫຼື ແອັບຂື້ນມາເພື່ອສະແດງ data ແລະ ເອົາ data ຕ່າງໆໄປນຳໃຊ້ຕາມຈຸດປະສົງຂອງເວັບ ແລະ ແອັບ...)
• ຄວາມຮູ້ທາງດ້ານ Network (ການເຊື່ອມຕໍ່ microcontroller ກັບ ອິນເຕິເນັດເພື່ອສົ່ງຄ່າຕ່າງໆ, ນຳເອົາ data ຈາກ sensor ເພື່ອສະແດງໃນໜ້າ dashboard ຫຼື ເອົາໄປວິເຄາະຕ່າງໆ...)
